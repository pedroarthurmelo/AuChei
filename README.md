# AuChei
Sistema com arquitetura robusta e segura para encontrar seu animalzinho perdido, Bem-vindo ao nosso projeto de faculdade (PUCPR - Cibersegurança)


## Estrutura JavaScript modular

- `data.js`: dados locais e utilitários de armazenamento.
- `utils.js`: helpers visuais e DOM.
- `api.js`: integração com PHP de autenticação.
- `common.js`: navbar, tema, menus e logout.
- `pets.js`: listagem, recompensas, contato e nova publicação.
- `profile.js`: tela de perfil.
- `settings.js`: tela de configurações.
- `auth.js`: login e registro.
- `admin.js`: painel administrativo.
- `main.js`: bootstrap e roteamento por página.
